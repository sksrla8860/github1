내용을 표현하는 태그(element)

내용을 분리할 때에는 문단과 문단으로 구분한다.
하지만 단순하게 내용을 줄바꿈을 할 경우에는 <br /> 이라는 태그를 사용.

특수글자를 사용시에는 &amp;

< 을 사용하려 할때에는 &lt; 태그 사용.

> 를 사용하려 할때에는 &gt; 태그 사용.

문단을 나타내는 p태그는 줄바꿈이 자동으로 되지 않는다.
띄어쓰기 하나만 적용되지 않는다. <span> 내용상 그룹을 따로 표현하는 태그

강조를 나타내는 태그
